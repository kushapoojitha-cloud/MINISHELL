📌 Project Title:

MiniShell – Custom Unix Shell Implementation in C

📖 Project Description

MiniShell is a custom implementation of a Unix-like command-line shell developed using the C programming language. The goal of this project is to understand
how operating systems handle command execution, process management, and user interaction at a low level.
This shell mimics the behavior of standard Linux shells (like bash) by allowing users to execute commands, manage processes, and interact with the system through 
a terminal interface.
The project demonstrates core concepts of system programming such as process creation, inter-process communication, signal handling, and command parsing.
